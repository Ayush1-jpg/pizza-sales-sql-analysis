🍕 Pizza Sales SQL Analysis

Project Overview

This project analyzes a pizza sales dataset using SQL to extract useful business insights such as revenue trends, top-performing pizzas, and category contribution.

Dataset Tables

- orders
- orders_details
- pizzas
- pizza_types

Key Business Questions Solved

1. Total revenue generated
2. Top 5 most ordered pizzas
3. Revenue contribution by pizza category
4. Top 3 pizzas by revenue within each category
5. Daily revenue trend

Tools Used

- MySQL
- SQL (Joins, Aggregations, Window Functions)

Key Insights

- Classic category contributes the highest revenue.
- Certain pizzas dominate sales within each category.
- Revenue trends help understand peak sales days.
